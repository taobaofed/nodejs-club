Node 地下铁沙龙 #2

开往春天的 Node.js

寒冬已逝，春之伊始。

我们欣喜地看到，在 Node.js 的世界里，越来越多的新鲜面孔，越来越多的创新想法。

如这春天中的万物，骚动着，勃发着，让 Node.js 社区充满了春的气息。

本次我们请来了四位嘉宾，和我们一同沉浸于魔都之春，感受 Node.js 之美。

动起来，莫辜负，这春光。

视频记录

基德 @淘宝

Node.js 源站的发展与挑战

在过去的一年里，Node.js 源站用几十台虚拟机支撑了淘宝数十亿 PV 的流量，在过程中解决了很多问题，比如：异地多活、缓存、文件分发等。在未来，有更多的挑战等待着我们面对，欢迎各位来与我们交流。

穆客 @淘宝

aliNode

alinode 是阿里云出品的 Node.js 应用服务解决方案，是一套基于社区 Node 改进的运行时环境和服务平台。在社区的基础上内建了强大的支持功能，帮助开发者迅速洞见性能细节，快速定位疑难杂症，直探问题根源。

云际 @钉钉

用 nw.js 高效构建桌面应用

近年，移动端大火，而桌面端略显落寞。随着 nw.js 等优秀解决方案的出现，桌面应用开发也迎来的新的春天。今天我们来聊一聊钉钉是如何使用了 nw.js 来构建桌面端应用，如何将体验做到与 Native 媲美。

严清 @Teambition

Redis 客户端及高级用途

目前 Teambition 绝大部分涉及 Redis 的业务都是采用 thunk-redis 客户端，比如消息，队列，基本数据缓存，基础工具等。如何更好的使用 Redis 客户端，如何能挖掘更多高级功能，是我们常思考的问题。
