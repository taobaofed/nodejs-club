Node 地下铁沙龙 #4

企业级 Node.js 应用

又是一年春暖花开的时节，Node地下铁迎来了第四场活动。

这次活动我们相聚在六朝古都——南京。

作为一个软件开发的名城，在企业级开发的产业非常庞大

而 Node.js 作为新兴的互联网开发技术，在企业级开发上能否占据一席之地？

我们这次邀请到几位同时有两方面开发经验的专家，跟我们分享他们的想法和思路。

快来一起相约玄武湖畔吧！

元 彦 @淘宝

深入 Node.js 组件化的 UI 渲染

本分享将介绍组件化的 UI 渲染以 React 为代表可以使 React 运行在 Node.js 进行渲染直出，解决了传统 SPA 应用的首屏性能低 与 SEO 弱两大问题，同时对比传统的模版化渲染方案，各自的利弊。最后会畅聊阿里开源的 Rax 在此方面的一些努力与探索。

黄一君 @途牛

如何定位线上 Node 进程的运行时错误——基于 V8 引擎的底层监控实践

Node.js 单进程的模型下，健壮性可以由PM2等守护工具最后兜底一把，但是对于出现的阻塞很多时候都是无能为力的。而且阻塞本身也离我们并不遥远：一个 ejs 模板中的正则贪婪匹配就能导致几乎所有可用进程的服务不可用。所以做了一些研究，来试图定位到线上进程的异常点。
